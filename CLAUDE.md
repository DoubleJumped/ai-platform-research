# CLAUDE.md — ai-platform-research

This repo is a research corpus on enterprise AI platforms (Microsoft Copilot/Cowork stack, ChatGPT Enterprise, Claude Enterprise/Cowork) and AI coding tools (GitHub Copilot, Claude Code, Codex): model availability × deployment surface × configuration, plus derived decks.

## Rules

1. **PUBLIC REPO.** Never write employer names, colleague names, or internal/confidential data. The generic profile ("Canadian Crown corporation, ~1,000 knowledge workers, ~15 heavy coders") is the maximum specificity allowed.
2. **Check `reference/INDEX.md` first** before web-searching a question the corpus may already answer.
3. **Stamp every volatile fact**: `(as of YYYY-MM-DD, source: url)`. Unstamped volatile claims are bugs.
4. **Supersede, don't append.** When a fact changes, rewrite the line and update the stamp; newer stamp wins on cross-file contradiction — fix the older file in the same session you notice it.
5. **One topic per file**; update `reference/INDEX.md` in the same action as creating/renaming/archiving a file. Keep files under ~150 lines unless flagged as a decision synthesis.
6. **Recommendations are preview paths.** No hard vendor decision exists; files that recommend a direction must say so in a status banner and never present a path as the settled pick.
7. Decks in `decks/` are derived artifacts — regenerate them from the reference files they cite, not from memory.
